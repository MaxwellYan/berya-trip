Rainy Day Plan

雨天備案

下雨時可以替換的宜蘭兩天一夜室內行程，包含博物館、觀光工廠、文創園區、親子體驗與休息點。

兩天一夜雨天備案路線

Day 1：頭城 / 礁溪 / 宜蘭市

上午：蘭陽博物館，先安排室內展覽，適合雨天慢慢逛。

中午：頭城或礁溪用餐，可安排咖啡廳或室內餐廳休息。

下午：潭酵天地觀光工廠，適合親子、拍照、買伴手禮。

傍晚：礁溪溫泉飯店 / 湯屋，雨天泡湯最穩。

晚上：礁溪市區晚餐，或回飯店休息。

Day 2：員山 / 五結 / 羅東

上午：金車噶瑪蘭威士忌酒廠，適合室內參觀與伴手禮採買。

中午：員山或宜蘭市區用餐。

下午：宜蘭傳藝園區或溪和水產觀光工廠，依同行成員選擇。

備選：若想走文青路線，可改去羅東文化工場。

晚上：羅東夜市或返程。

室內備案

蘭陽博物館

類型：博物館 / 室內展覽 / 宜蘭文化。

地址：宜蘭縣頭城鎮青雲路三段 750 號。

適合替換：Day 1 上午 / Day 1 下午。

優點：雨天穩定、室內展覽完整，也適合作為宜蘭旅遊第一站。

備註：週三通常休館，出發前要確認開館資訊。

潭酵天地觀光工廠

類型：觀光工廠 / 室內景點 / 親子景點 / 伴手禮。

地址：宜蘭縣礁溪鄉漳福路 25 號。

適合替換：Day 1 下午 / 下雨時的臨時備案。

優點：免門票、停車相對方便、室內空間多，適合短暫停留。

備註：適合搭配礁溪住宿或礁溪晚餐。

金車噶瑪蘭威士忌酒廠

類型：酒廠參觀 / 室內導覽 / 伴手禮 / 咖啡休息。

地址：宜蘭縣員山鄉員山路二段 326 號。

適合替換：Day 2 上午 / Day 2 下午。

優點：不怕下雨、動線舒服，適合大人行程與伴手禮採買。

備註：若有開車，試飲需注意駕駛安全。

宜蘭傳藝園區

類型：傳統藝術 / 室內展館 / 老街場景 / 手作體驗。

地址：宜蘭縣五結鄉五濱路二段 201 號。

適合替換：Day 2 下午 / 原本戶外景點取消時。

優點：園區大、內容多，雨小可以走街區，雨大可改逛展館與商店。

備註：部分區域仍需戶外移動，建議帶傘。

溪和水產觀光工廠

類型：觀光工廠 / 食育體驗 / 海鮮料理 / 親子景點。

地址：宜蘭縣五結鄉利興二路 11 號。

適合替換：Day 2 中午 / Day 2 下午。

優點：可吃、可逛、可買伴手禮，停留時間彈性。

備註：通常週三休館，行程前要再確認。

羅東文化工場

類型：文創展覽 / 建築空間 / 室內展演 / 雨天散步。

地址：宜蘭縣羅東鎮純精路一段 96 號。

適合替換：Day 2 下午 / 羅東周邊備案。

優點：靠近羅東市區，適合接羅東夜市或返程前短暫停留。

備註：週一及每月最後一日通常休館，需留意公告。

雨天行程替換建議

如果 Day 1 下大雨

建議路線：蘭陽博物館 → 礁溪午餐 → 潭酵天地 → 礁溪泡湯 / 飯店休息。

適合對象：親子、長輩、想要輕鬆玩的人。

優點：移動距離不會太遠，雨天也不會太狼狽。

如果 Day 2 下大雨

建議路線：金車噶瑪蘭威士忌酒廠 → 宜蘭市區午餐 → 傳藝園區 / 溪和水產 → 羅東夜市。

適合對象：朋友、情侶、家庭旅遊。

優點：室內點多，可依天氣彈性縮短或延長停留時間。
